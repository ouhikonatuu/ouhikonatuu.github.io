HTML

发表于 2022-07-01

什么是HTML

  • 超文本标记语言(Hyper Text Markup Language)，是一种标记语言。
  • 表现形式为一系列标签
  • 可以将网络上的文档格式统一
  • 可以使分散的internet资源连接成为一个逻辑整体

表现形式

  • 双标签、单标签

双标签

  • <标签名></标签名>
  • 双标签分为开放&闭合 || 开始&结束 || 头&尾 标签
  • 标签名小写

单标签

  • <标签名 />
  • 单标签无开放标签，需要闭合标签
  • 标签名小写

属性

  • <a href="https://github.com/"></a>
  • <img src="https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png" />
  • 标签名加空格后书写
  • 属性名 = “属性值”

结构

<!DOCTYPE html>

  • 设置html版本
  • HTML5的声明方式
  • 兼容性模式
    • CSS1Compat(声明后)
      • W3C标准兼容性模式
    • BackConpat
      • 浏览器怪异兼容性模式

<html></html>

  • html根标签
  • 属性lang：设置语言类别

<head></head>

  • 向浏览器传达网页的基本信息和配置
<title></title>
  • 内容在网页标题中显示，搜索引擎检索权重最高。四十字以内
  • 主页：网站名称 + 主要关键字/关键词的描述
  • 详情页：详情名称 + 网站名称 + 简介
  • 列表页：分类名称 + 关键字 + 网站名称
  • 文章页：标题 + 分类 + 网站名称

<meta name="keywords content="关键字" />

  • 网站的关键字，100个字符，小写逗号分隔
  • 网站名称 + 分类信息 + 网站名称

<meta name="description content="描述" />

  • 网站的表述信息，权重次高，120字以内
  • 综合title + keywords的简单描述

<meta charset="" />

  • 属性charset：设置编码字符集

<body></body>

  • 页面主体结构，内容展示区域
