JavaScript

发表于 2022-07-01

什么是JavaScript

  • ECMAscript、DOM(decument object model)、BOM(browser object model)
  • JavaScript 为动态语言 -> 脚本语言 -> 解释型语言 -> 弱类型语言(弱化数据类型)
  • 如 Java 为静态语言 -> 编译型语言 -> 强类型语言(强制数据类型)

轮转时间片

  • 单线程模拟多线程
  • 短时间之内轮流执行多个任务的片段
  • 随机排列任务被切分后的片段，组成队列
  • JS线程依次执行任务片段

<script></script>

  • src属性（可选）：引用外部.js文件

变量

  • 声明变量：var -> variable，var 变量名
  • 赋值：变量名 = 值/表达式
  • 命名规范：以字母、下划线、$开头且含语义的非关键字/保留字命名，驼峰格式

值

  • 原始值（基本类型）：Number、 String、 Boolean、 undefined、 null
  • 引用值（引用类型）：Object、 Array、 Function、 Date、 RegExp

栈与堆

  • 栈内存：存放原始值或引用值指向堆内存地址的指针，不可改变，赋值会开辟新空间
  • 堆内存：存放引用值，可修改，赋值会开辟新空间并传递新地址

错误

  • 语法问题：代码块不执行

  • 通用问题：中断代码执行

运算

  • +、 -、 *、 /、 %、 ()、 >>、 <<
  • Number类型：Nan、 (-)infinity（无穷）

比较

  • >、 <、 >=、 <=、 ==、 ===、 !=、 !==
  • 字符串比较：相对应的ASCII码，从左到右依次比较
  • ===校验数据类型，==不校验
  • !==校验数据类型，!=不校验
  • NaN与包括自己的任何值不相等，Nan != NaN
  • 返回结果为布尔类型

分支控制

  • if(){}else if(){}else{}
  • 范围数据判断时使用，条件互斥时，使用else if 减少判断次数，使用else处理边界问题
  • switch(){case: break;default:;}
  • 离散值判断时使用，性能优越，default处理边界问题

逻辑运算

  • 与
    • 逻辑与：&&，前置条件执行并判断为false，终止运行，返回false；前后条件执行并判断后返回true，后置条件优先
    • 按位与：&，前后条件都会执行并判断
  • 或
    • 逻辑或：||，前置条件执行并判断为true，终止运行，返回true；前后条件执行并判断后返回false，后置条件优先
    • 按位或，前后条件都会执行并判断
  • 非
    • 逻辑非：！，返回与条件对应的相反布尔值；
    • 按位非：~，将二进制每位取反
  • 默认为false的类型：undefined、 null、 NaN、 “”、 0

循坏

  • 先判断执行条件，为true则执行表达式，反之终止循环
  • for(变量(选填); 判断条件; 表达式){执行语句}
  • while(判断条件){执行语句}
  • do{执行语句}while(判断条件)
    • 首次执行语句的判断条件强制为true
